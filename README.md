# Front-end App for the "Ts API"

This API was built using Eslint and Typescript for a better code reading and safety.

## Installation ✨

Use the package yarn to install the dependecies.

```bash
yarn install
```

## Usage 🔥

```javascript
yarn start
```

## Tools used in this project 🍏

```javascript
Eslint
Styled-components
React-router
Hooks
```
