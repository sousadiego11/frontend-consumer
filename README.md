# Front-end App that consumes external API ( WIP )

This project was built using ReactJS and Mapbox(react-map-gl).

## Installation ✨

Use the package yarn to install the dependecies.

```bash
yarn install
```

## Usage 🔥

```javascript
yarn start
```

## Tools used in this project 🍏

```javascript
Eslint
Styled-components
React-router
Hooks
Mapbox
```
